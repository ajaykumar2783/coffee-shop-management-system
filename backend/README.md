# 🚀 DevDuo.in – Portfolio Website

> 💡 A modern, responsive, and premium portfolio website built by **Ajay & Raghu** to showcase projects, skills, and services.

---

## 🌐 Live Demo
🔗 https://devduo-in.vercel.app/

---

## ✨ Features

- ⚡ Modern UI with **glassmorphism & gradients**
- 🎯 Fully **responsive design (mobile + desktop)**
- 🌙 **Dark mode toggle**
- ✍️ Animated typing text
- 🎨 Premium **hover animations**
- 📊 Skill progress bars
- 🧠 Project filtering system
- 📈 Counter animations
- 🧭 Smooth scroll & reveal animations
- 🔝 Back-to-top button
- 📬 Contact form (email integration)

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Icons:** Font Awesome  
- **Fonts:** Google Fonts (Inter)  
- **Deployment:** GitHub Pages / Netlify / Vercel
